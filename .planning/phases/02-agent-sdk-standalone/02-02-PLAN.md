QueuedTask dataclass fields:

- channel: str

- notify_callback: callable  # async — called when task starts: "Your task is now running."

Module-level state (private):
- `_queue: asyncio.Queue` — initialized as asyncio.Queue(maxsize=TOTAL_SLOTS) in `initialize()` function
- `_current_task: QueuedTask | None`
- `_running_asyncio_task: asyncio.Task | None` — the asyncio.Task wrapping the agent call, stored for cancel

IMPORTANT: Do NOT initialize `_queue` at module import time with `asyncio.Queue()` — asyncio event loop may not exist yet. Instead, provide an `initialize()` function that creates the queue, and have `run_queue_loop()` call it if not already initialized. Or create _queue inside run_queue_loop() before the while loop. Either approach works; choose the simpler one.

Public API:

`def enqueue(task: QueuedTask) -> bool`:
- try: _queue.put_nowait(task); return True
- except asyncio.QueueFull: return False
- If _queue is None (not initialized): raise RuntimeError("Queue not initialized — call run_queue_loop() first")

`def is_full() -> bool`: return _queue is not None and _queue.full()

`def queue_depth() -> int`: return _queue.qsize() if _queue else 0

`def get_current_task() -> QueuedTask | None`: return _current_task

`def get_state() -> dict`:
Returns observable snapshot for /sb-status:
```python
{
    "current": _current_task,      # QueuedTask or None
    "queue_depth": queue_depth(),  # int 0-3
    "is_full": is_full(),          # bool
}
```

`def cancel_running() -> bool`:
- If _running_asyncio_task is None or done: return False
- _running_asyncio_task.cancel(); return True
- Note: cancellation is cooperative — the queue loop catches CancelledError and treats it like a timeout

`async def run_queue_loop() -> None`:
Long-running coroutine. Must be started once at bot startup with asyncio.create_task().
```python
async def run_queue_loop():
    global _queue, _current_task, _running_asyncio_task
    _queue = asyncio.Queue(maxsize=TOTAL_SLOTS)
    while True:
        task = await _queue.get()
        _current_task = task
        try:
            await task.notify_callback()
            coro = run_agent_with_timeout(task.prompt, task.session_id)
            _running_asyncio_task = asyncio.ensure_future(coro)
            result = await _running_asyncio_task
            await task.result_callback(result)
        except asyncio.CancelledError:
            # /cancel triggered — treat as user-initiated cancellation
            result = {
                "session_id": task.session_id,
                "result": None,
                "subtype": "error_cancelled",
                "num_turns": -1,
                "partial_texts": [],
            }
            await task.result_callback(result)
            # Do NOT re-raise — let queue loop continue to next task
        except Exception as exc:
            log.exception("queue_loop: unexpected error", error=str(exc))
            await task.result_callback({
                "session_id": task.session_id,
                "result": str(exc),
                "subtype": "error_internal",
                "num_turns": -1,
                "partial_texts": [],
            })
        finally:
            _current_task = None
            _running_asyncio_task = None
            _queue.task_done()
```

Critical: catch `Exception` not bare `except` inside the loop body, so the loop survives individual task failures (RESEARCH.md Pitfall 2). Let CancelledError propagate from the outer `asyncio.ensure_future` task, then catch it explicitly as shown above.

```python
def split_long_message(text: str, max_chars: int = 3800) -> list[str]:
    if len(text) <= max_chars:
        return [text]
    chunks = []
    current = []
    current_len = 0
    for line in text.splitlines(keepends=True):
        if current_len + len(line) > max_chars and current:
            chunks.append("".join(current))
            current = []
            current_len = 0
        current.append(line)
        current_len += len(line)
    if current:
        chunks.append("".join(current))
    return chunks
```

def format_status(current: Optional[dict], recent: list[dict], uptime: str, queue_snapshot: dict = None) -> str:
    lines = ["*SuperBot Status*"]
    if current:
        elapsed = int(time.time() - current["started_at"])
        lines.append(f"Running ({elapsed}s): {current['text'][:100]}")
    elif queue_snapshot and queue_snapshot.get("current_task"):
        qt = queue_snapshot["current_task"]
        lines.append(f"Running: {qt.prompt[:100]}")
    else:
        lines.append("Idle")
    if queue_snapshot:
        depth = queue_snapshot.get("queue_depth", 0)
        if depth > 0:
            lines.append(f"Queued: {depth} task(s) waiting")
    if recent:
        lines.append(f"Recent ({len(recent)}):")
        for t in recent:
            lines.append(f"  - {t['text'][:80]}")
    lines.append(f"Uptime: {uptime}")
    return "\n".join(lines)

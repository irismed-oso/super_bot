Step B — Create scripts/test_agent.py:

The harness must:
1. Accept a positional `prompt` argument
2. Accept `--thread-ts` (default: "test-thread-001") for session keying
3. Accept `--max-turns` (default: 25) to override module default
4. Accept `--timeout` (default: 600) to override module default
5. Accept `--channel` (default: "C_TEST") — needed for session_map key
6. Look up existing session_id from session_map using (channel, thread_ts)
7. Print: "Session: new" or "Session: resuming {session_id[:16]}"
8. Call run_agent_with_timeout(prompt, session_id, timeout_seconds=args.timeout, max_turns=args.max_turns)
9. If result["session_id"] is not None: call session_map.set(channel, thread_ts, result["session_id"])
10. Print the separator line, result subtype, num_turns
11. Print result["result"] if present, or "(no result captured)"
12. If subtype == "error_max_turns": print "Reply with --thread-ts {args.thread_ts} to continue where I left off."
13. If subtype == "error_timeout": print "Timed out. Reply with --thread-ts {args.thread_ts} to continue."
14. If subtype == "error_cancelled": print "Cancelled."
15. Show partial_texts if present and result is None (timeout case)

Use `if __name__ == "__main__": asyncio.run(main())` pattern.

Add a docstring at the top matching the usage examples from RESEARCH.md Pattern 6:
```
Usage:
    python scripts/test_agent.py "list files in the bot/ directory"
    python scripts/test_agent.py "what did you just find?" --thread-ts 12345.67890
    python scripts/test_agent.py "count from 1 to 100" --max-turns 2
    python scripts/test_agent.py "sleep for 15 minutes" --timeout 5
```

Ensure scripts/ directory exists (create with os.makedirs if not present, or just create the file — Python will error if dir missing).

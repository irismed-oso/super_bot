```python
import asyncio
import os
import structlog
from claude_agent_sdk import query, ClaudeAgentOptions, AssistantMessage, ResultMessage, TextBlock

log = structlog.get_logger(__name__)

# Constant CWD — must match on every call or session resume silently starts fresh (Pitfall 1 in RESEARCH.md)
MIC_TRANSFORMER_CWD = os.path.realpath(
    os.environ.get("MIC_TRANSFORMER_CWD", "/home/bot/mic_transformer")
)

TIMEOUT_SECONDS = 600   # 10 minutes — locked decision (CONTEXT.md Safety Limits)
MAX_TURNS = 25          # locked decision (CONTEXT.md Safety Limits)
```

Implement `run_agent(prompt, session_id, *, on_text=None) -> dict`:
- Creates ClaudeAgentOptions(cwd=MIC_TRANSFORMER_CWD, resume=session_id, max_turns=MAX_TURNS, permission_mode="bypassPermissions")
- Iterates `async for message in query(prompt=prompt, options=options):`
  - If isinstance(message, AssistantMessage): extract TextBlock texts, call await on_text if provided, accumulate in partial_texts list
  - If isinstance(message, ResultMessage): capture message.session_id, message.result, message.subtype, message.num_turns
- Returns dict: {"session_id": ..., "result": ..., "subtype": ..., "num_turns": ..., "partial_texts": [...]}
- Log with structlog at start (prompt[:80], session_id) and end (subtype, num_turns)

Implement `run_agent_with_timeout(prompt, session_id, *, on_text=None, timeout_seconds=TIMEOUT_SECONDS, max_turns=MAX_TURNS) -> dict`:
- Override MAX_TURNS on ClaudeAgentOptions if max_turns differs from constant (for test harness --max-turns flag)
- Wraps run_agent() in asyncio.wait_for(..., timeout=timeout_seconds)
- On asyncio.TimeoutError: log warning, return {"session_id": session_id, "result": None, "subtype": "error_timeout", "num_turns": -1, "partial_texts": []}
- Note: retain the prior session_id on timeout — caller can still resume (RESEARCH.md Pattern 2)

Key constraints:
- Do NOT use continue_conversation=True — always explicit resume= (RESEARCH.md Anti-Patterns)
- session_id=None is valid for a new session — passes None to resume= which SDK treats as "new"
- max_turns param on run_agent_with_timeout overrides the module constant (needed by test harness)
- partial_texts list in the return dict lets callers show "partial output captured" on timeout (RESEARCH.md Open Question 1)

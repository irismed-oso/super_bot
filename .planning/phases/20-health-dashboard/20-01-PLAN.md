**Imports to add at top of file:**

**Handler implementation (`_handle_bot_health`):**

Gather these metrics and format as a compact emoji-prefixed list:

1. **Status line**: Current task or "Idle" -- from `queue_manager.get_state()["current"]`
2. **Uptime**: from `task_state.get_uptime()`
3. **Queue**: depth from `queue_manager.get_state()["queue_depth"]`
4. **Version**: git short SHA via `subprocess.check_output(["git", "rev-parse", "--short", "HEAD"], text=True).strip()` -- wrap in try/except, fallback to "unknown"
5. **Memory**: RSS in MB via `resource.getrusage(resource.RUSAGE_SELF).ru_maxrss` (on macOS this is bytes, on Linux it's KB -- divide accordingly using `sys.platform`). Import `resource` and `sys`.
6. **Disk**: Use `shutil.disk_usage("/")` to show used/total in GB. Import `shutil`.
7. **CPU**: Skip -- not feasible without psutil on a 2 GB VM. Omit per Claude's discretion.
8. **Recent tasks**: Count of last 5 tasks from `task_state.get_recent(5)`, showing success count (all recent tasks are implicitly successful since failures don't enter recent list)
9. **Active monitors**: Count from `background_monitor.get_active_monitors()`, with brief labels if any active
10. **Errors (24h)**: Run `subprocess.check_output(["journalctl", "-u", "superbot", "--since", "24 hours ago", "-p", "err", "--no-pager", "-q"], text=True, timeout=5)` and count non-empty lines. Wrap in try/except -- fallback to "unavailable" (won't work on macOS dev, only on VM). Use `stderr=subprocess.DEVNULL` to suppress journal warnings.
11. **Last restart**: Format `task_state._start_time` as ISO timestamp using `datetime.fromtimestamp()`. Import `datetime` from `datetime`.
